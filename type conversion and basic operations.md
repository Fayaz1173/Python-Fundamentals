# 1 String to Integer Conversion

### ➤ Program

```
num_str="25"
num_int=int(num_str)

print("String value:",num_str)
print("After conversion:",num_int)
print("Addition result:",num_int+5)
```

### ➤ Output

```
String value: 25
After conversion: 25
Addition result: 30
```

# 2 Integer to Float Conversion

### ➤ Program

```
num=10
num_float=float(num)

print("Integer:",num)
print("Converted to float:",num_float)
print("Division result:",num_float/4)
```

### ➤ Output

```
Integer: 10
Converted to float: 10.0
Division result: 2.5
```

---

# 3 Float to Integer Conversion

### ➤ Program

```
price=99.99
price_int=int(price)

print("Original float:",price)
print("Converted to int:",price_int)
```

### ➤ Output

```
Original float: 99.99
Converted to int: 99
```

# 4 Integer to String Conversion

### ➤ Program

```
age=21
age_str=str(age)

print("My age is "+age_str)
```

### ➤ Output

```
My age is 21
```

# 5 Basic Arithmetic Operations

### ➤ Program

```
a=10
b=3

print("Addition:",a+b)
print("Subtraction:",a-b)
print("Multiplication:",a*b)
print("Division:",a/b)
print("Floor Division:",a//b)
print("Modulus:",a%b)
print("Power:",a**b)
```

### ➤ Output

```
Addition: 13
Subtraction: 7
Multiplication: 30
Division: 3.3333333333333335
Floor Division: 3
Modulus: 1
Power: 1000
```

# 6 Type Checking

### ➤ Program

```
x=5
y=5.0
z="5"

print(type(x))
print(type(y))
print(type(z))
```

### ➤ Output

```
<class 'int'>
<class 'float'>
<class 'str'>
```

---

# 7 Boolean Conversion

### ➤ Program

```
print(bool(0))
print(bool(1))
print(bool(""))
print(bool("Python"))
```

### ➤ Output

```
False
True
False
True
```

---

# Important Points

- `int()` → converts to integer
- `float()` → converts to decimal
- `str()` → converts to string
- `bool()` → converts to True/False
- `type()` → checks data type
